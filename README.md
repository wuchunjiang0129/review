### UAV+VLN
##  AerialVLN: Vision-and-Language Navigation for UAVs

**论文链接：**[arXiv:2308.06735](https://arxiv.org/abs/2308.06735)（ICCV 2023）

**仓库链接：**[AirVLN/AirVLN](https://github.com/AirVLN/AirVLN)

**关键词：** 空中视觉语言导航（Aerial VLN）、无人机（UAV）、多模态学习、具身智能、连续环境、跨模态对齐、AirSim

**简要概述：**本文提出面向无人机的空中视觉语言导航任务 AerialVLN，要求智能体依据自然语言指令和第一视角 RGB、深度观测，在无预建导航图的连续三维城市环境中完成长距离导航与避障。作者基于 Unreal Engine 4 和 AirSim 构建了包含 25 个场景、8,446 条航迹及 25,338 条指令的数据集，并提出 Look-ahead Guidance，通过兼顾返回示范航迹和沿指令路线前进来改善偏航后的动作监督。实验中 CMA 在完整测试未见场景的成功率仅为 1.6%，远低于人工的 80.8%，说明长时序决策、三维空间推理、跨模态地标对齐、偏航恢复和准确停止仍是该任务的核心难点。
